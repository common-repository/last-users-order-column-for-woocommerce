== Description ==

Plugin that allows you to easily see last order for a user in WordPress user list by adding additional column called Last Order.

== Installation ==

= Minimum Requirements =

* WordPress 4.0 or greater
* WooCommerce 4.0 or greater

= Setup =

This section describes how to install OnSale Page for WooCommerce plugin and get it working.

1. Upload the plugin files to the /wp-content/plugins/ directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
8. Go to WordPress user list.
